- `useNavigation.ts` - Navegación entre vistas

## 📦 Instalación

```bash
# Clonar el repositorio
git clone https://github.com/JereGuale/StreamZone.git

# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev

# Build para producción
npm run build
```
